# 📌 Project: Bank Data Processing

## 🎯 Цель проекта

Проект предназначен для обработки банковских данных:
	•	маскировка номеров карт и счетов;
	•	фильтрация операций по статусу (EXECUTED, CANCELED и др.);
	•	сортировка операций по дате;
	•	преобразование даты в удобочитаемый формат;
	•	работа с JSON, CSV и XLSX файлами с транзакциями;
	•	конвертация валют через внешний API;
	•	поиск транзакций по описанию с помощью регулярных выражений;
	•	подсчёт количества операций по категориям;
	•	логирование работы функций и модулей.

---

## 📂 Описание проекта

### Модули проекта

	•	masks — маскирование карт и счетов;
	•	widget — работа с данными карт/счетов и форматирование даты;
	•	processing — фильтрация и сортировка операций;
	•	utils — чтение транзакций из JSON;
	•	file_reader — чтение транзакций из CSV и XLSX;
	•	external_api — конвертация валют с использованием API;
	•	decorators — декораторы для логирования функций;
	•	generators — генераторы для работы с транзакциями и номерами карт;
	•	search — поиск транзакций по описанию и подсчёт операций по категориям;
	•	main — основной модуль, связывающий функциональности между собой.

# 🏦 Анализ банковских транзакций

## 📌 Описание проекта
Приложение для анализа транзакций из Excel/CSV/JSON-файлов.  
Позволяет работать с данными, фильтровать, анализировать, формировать отчёты и предоставлять сервисы.  

Проект реализован на **Python 3.13**, использует **Poetry** для управления зависимостями.  
Ключи для API скрыты в файле `.env`.  

---

## ⚙️ Функционал проекта

### 🔍 Поиск и обработка
- Поиск транзакций по описанию с помощью `re`.
- Подсчёт количества операций по категориям.
- Маскирование номеров карт и счетов.
- Преобразование дат транзакций в удобный формат.

### 🖥️ Веб-страницы
- **Главная (`views.py`)**
  - Приветствие в зависимости от времени суток.
  - Сумма расходов и кешбэк по каждой карте.
  - Топ-5 транзакций по сумме.
  - Курс валют.
  - Цены на акции (S&P500).

- **События (`views.py`)**
  - Сумма расходов и доходов.
  - Аналитика по категориям.
  - Данные по переводам и наличным.

### 🛠️ Сервисы (`services.py`)
- **Инвесткопилка** — округление транзакций до заданного лимита (10, 50, 100 ₽).  
  Пример: трата 1712 ₽ округляется до 1750 ₽, разница (38 ₽) отправляется в копилку.
- **Простой поиск** — поиск транзакций по ключевым словам.
- **Поиск по телефонным номерам** — находит переводы с номерами в описании.
- **Поиск переводов физическим лицам** — извлекает переводы с упоминанием имён.

### 📊 Отчёты (`reports.py`)
- **Траты по категории** — сумма за последние 3 месяца.
- **Траты по дням недели** — средние расходы по дням недели.
- **Траты в рабочие/выходные дни** — сравнение трат.

Отчёты автоматически сохраняются в файлы через декоратор.
